提示自动合并两处，一处有冲突，选中左边的有冲突的行点工具栏中的复制到右侧（Alt+Right）以左边的修改准，如果想以右边的修改为准备则选中右国的有冲突的行点工具栏中的复制到左侧（Alt+Left）。

如果都不对，那么直接在中间进行修改。完成后保存并关闭winmerge。完成冲突解决，如果你的冲突文件同级目录下出现了*.bak的文件，那么需要在winmerge中配置关闭自动备份功能。
